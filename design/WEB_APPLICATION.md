# Web Application Design Guidelines

## URL Design & Routing

### Slug Naming Conventions
**Rule**: All URL slugs must be **lowercase**.

**Correct:**
- `/provider/antigravity`
- `/provider/claude-3-5-sonnet`

**Incorrect:**
- `/provider/Antigravity`
- `/provider/Claude-3.5-Sonnet`

**Rationale:**
- **Consistency**: Lowercase URLs are a standard convention in web development.
- **Case Sensitivity**: URLs are often case-sensitive; enforcing lowercase avoids duplication and confusion.
- **Aesthetics**: Lowercase URLs look cleaner and more professional.
