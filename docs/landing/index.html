Features

Everything you need to stay on top of your AI usage

🔍

Smart Discovery

Automatically scans environment variables, Claude Code credentials, and standard config files. Zero configuration friction.

📊

Live Dashboard

Compact always-on-top window with real-time progress bars showing remaining credits, quotas, and spending.

📢

Tray Integration

Per-provider tray icons with live progress bars. Glance at your system tray to see which providers are healthy.

⚡

Pace Projection

Will you hit your limit before reset? Headroom, on-pace, and over-pace badges tell you at a glance.

🌐

Web Dashboard

Browse usage history, charts, and analytics from any browser. Seven built-in themes including Dracula and Nord.

🔃

Auto-Updates

In-app release notifications with one-click download. Beta and stable channels supported.
