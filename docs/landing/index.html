Features

Everything you need to stay on top of your AI usage

Desktop GUI

  • Live progress bars — remaining credits, quotas, and spending at a glance
  • System tray — per-provider icons with live bars, auto-hide on focus loss
  • Pace projection — predicts if you'll hit your limit before reset
  • Card presets — default, compact, detailed, and dual-bar modes
  • Auto-updates — beta and stable channels with one-click install

Web Dashboard

  • Full dashboard — spending summaries, burn-rate forecasts, sparklines
  • Interactive charts — time-series with server-side downsampling
  • Per-model analytics — breakdown tables, latency trends, raw JSON
  • Reliability monitoring — uptime, response times, error rates
  • 14 themes — Dracula, Nord, Catppuccin, Monokai, and more

CLI

  • Status & history — check usage from any terminal
  • Key management — set, remove, and scan for API keys
  • Export — CSV and JSON, up to 90 days
  • Monitor control — start, stop, restart the background service
  • Auto-discovery — scans env vars and config files for existing keys
